# 数据统计

站点数据总览、增长趋势、排行榜等统计接口，所有端点均需 `stats:read` 或 `system:read` scope。

---

## 站点数据总览

<Endpoint method="POST" path="openApi.overview" auth="stats:read" />

获取全站关键指标一览，包含用户数、内容数、浏览量、互动统计等。

<CopyBlock>{`curl -X POST 'https://your-domain.com/api/trpc/openApi.overview' \\
  -H 'Content-Type: application/json' \\
  -H 'Authorization: Bearer sk-your-api-key' \\
  -d '{}'`}</CopyBlock>

---

## 区间增长统计

<Endpoint method="POST" path="openApi.growth" auth="stats:read" />

查询指定日期范围内的新增数据。

<ParamTable>
  <P name="from" type="string" required>开始日期（ISO 8601 格式）</P>
  <P name="to" type="string" required>结束日期（ISO 8601 格式，范围不超过 90 天）</P>
</ParamTable>

---

## 增长趋势

<Endpoint method="POST" path="openApi.growthTrend" auth="stats:read" />

按日聚合的趋势序列，返回每日的用户注册、内容发布、浏览、互动等数据。

<ParamTable>
  <P name="from" type="string" required>开始日期</P>
  <P name="to" type="string" required>结束日期</P>
</ParamTable>

---

## 内容排行榜

<Endpoint method="POST" path="openApi.leaderboard" auth="stats:read" />

<ParamTable>
  <P name="type" type="string" required>内容类型：`video` / `game` / `image`</P>
  <P name="metric" type="string" required>排行指标：`views` / `likes` / `favorites` / `comments`</P>
  <P name="limit" type="number">数量（1-100，默认 20）</P>
</ParamTable>

<ParamTable>

## 用户注册统计

<Endpoint method="POST" path="openApi.userStats" auth="stats:read" />
